## Requisitos

- [Raylib](https://github.com/raysan5/raylib)

## Como executar

Se usa o GCC, apenas use o comando `make` no diretório do programa e execute o arquivo `snake.exe` ou use o comando `make teste`.

Caso use outro compilador, compile manualmente.

## Sobre o projeto

O programa implementa o clássico snake game na linguagem C e usando a Raylib para os gráficos.
Por exigencia do professor, a cobra é armazenada usando uma lista encadeada.
